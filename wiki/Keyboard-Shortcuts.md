# Keyboard Shortcuts

> Keyboard chords surfaced by `KeyboardShortcutsHelper`.
>
> *Audience: end user · Last reviewed: 2026-05-02*

The shortcut helper (press **?** anywhere outside an input) lists
every chord. The table below is the current set; check the helper
for any additions added after the "Last reviewed" date above.

---

## Global

| Chord | Action |
| --- | --- |
| `?` | Open the shortcuts helper |
| `Ctrl/Cmd + K` | Open command palette / quick-switch |
| `Ctrl/Cmd + ,` | Open Settings |
| `Esc` | Close the topmost dialog / sheet |
| `Ctrl/Cmd + /` | Toggle theme |

## Tab navigation

| Chord | Action |
| --- | --- |
| `1`–`9` | Jump to tab N |
| `Ctrl/Cmd + Tab` | Next tab |
| `Ctrl/Cmd + Shift + Tab` | Previous tab |

## Chat

| Chord | Action |
| --- | --- |
| `Ctrl/Cmd + N` | New conversation |
| `Ctrl/Cmd + F` | Search conversations |
| `Ctrl/Cmd + Enter` | Send message |
| `Shift + Enter` | Newline in the input |
| `Ctrl/Cmd + Up` | Edit your last message |
| `R` (when message focused) | Regenerate |

## Agents

| Chord | Action |
| --- | --- |
| `Ctrl/Cmd + Shift + N` | New agent |
| `Space` (when agent focused) | Run agent |
| `Ctrl/Cmd + .` | Stop running agent |

## Workflows

| Chord | Action |
| --- | --- |
| `Ctrl/Cmd + S` | Save workflow |
| `Ctrl/Cmd + Z` / `Shift + Ctrl/Cmd + Z` | Undo / redo |
| `Delete` | Remove selected node / edge |
| `Ctrl/Cmd + D` | Duplicate selected node |

## IDE

| Chord | Action |
| --- | --- |
| `Ctrl/Cmd + S` | Save file (auto-save also runs) |
| `Ctrl/Cmd + P` | Quick-open file |
| `Ctrl/Cmd + Shift + F` | Find in project |

---

## Mobile

Most shortcuts are hidden on mobile (no physical keyboard).
External-keyboard users can still use them via Bluetooth keyboards.

The hardware **back** button on Android is wired to the back-handler
stack — see [Mobile & Android](Mobile-and-Android#back-button-handling).

---

## Customizing

There is no in-app shortcut editor today. The current set lives in
the components themselves; PRs that change a chord must update this
page.
