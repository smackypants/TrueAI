## Mandatory attribution (for forks and derivatives)

Beyond MIT's basic notice requirement, this project requires:

1. The
   [`LICENSE`](https://github.com/smackypants/TrueAI/blob/main/LICENSE)
   file is preserved unmodified.
2. The
   [`NOTICE`](https://github.com/smackypants/TrueAI/blob/main/NOTICE)
   file is preserved unmodified.
3. All in-source copyright headers
   (`Copyright (c) 2024-2026 Skyler Jones ("smackypants") /
   Advanced Technology Research`) are preserved.
4. Visible credit is given to the original author in any forked
   distribution, app store listing, or derivative work.

PRs that strip headers, or fork releases that omit the NOTICE, are
out of compliance.
